## 模型列表接口增加 modality 筛选

`GET /v1/models?modality=image` 可按分类筛选模型。修改文件：
- `src/routes/models.ts` - handleListModels 接受 request 参数，解析 modality 查询参数进行过滤；响应中每个 model 新增 modality 字段
- `src/index.ts` - 调用时传入 request
有效 modality 值：text、image、video、file；不传则返回全部。

## 模型新增"价格"字段

模型创建和编辑时新增 `price` 字段（可选字符串类型），修改文件：
- `src/types.ts` - `UpstreamModelConfig` 和 `ModelConfig` 接口新增 `price?: string`
- `src/config.ts` - `listModels()` 在构建 `ModelConfig` 时传播 `price` 字段
- `src/routes/admin.ts`:
  - `normalizeModelInput()` - 解析 price 输入
  - HTML 表单 - 新增"价格（元/1M Token）"输入框 `#model-price`
  - `saveModelFromForm()` - 提交时包含 price
  - `fillModelEditor()` / `resetModelForm()` / `toModelInput()` - 读写 price
  - 模型列表渲染和详情展示 - 显示价格信息
  - 上游详情模型表格 - 新增价格列
  - `publicModel()` - 响应中包含 price 字段
