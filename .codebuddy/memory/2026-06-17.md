# 2026-06-17

## 新增功能：任务列表 API
- 新增 `GET /v1/tasks` 端点，用户可以通过 API 查看自己的任务列表
- 支持分页参数：`?limit=25`（1-100，默认25）和 `?after=task_id`（游标分页）
- 返回格式：`{ object: "list", data: [...], has_more: boolean, first_id, last_id }`
- 自动按 `organization_id` 过滤，用户只能看到自己组织的任务
- 修改文件：`src/routes/tasks.ts`（新增 `handleListTasks`）、`src/index.ts`（新增路由）
