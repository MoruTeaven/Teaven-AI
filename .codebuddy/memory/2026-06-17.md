# 2026-06-17

## 新增功能：任务列表 API
- 新增 `GET /v1/tasks` 端点，用户可以通过 API 查看自己的任务列表
- 支持分页参数：`?limit=25`（1-100，默认25）和 `?after=task_id`（游标分页）
- 返回格式：`{ object: "list", data: [...], has_more: boolean, first_id, last_id }`
- 自动按 `organization_id` 过滤，用户只能看到自己组织的任务
- 修改文件：`src/routes/tasks.ts`（新增 `handleListTasks`）、`src/index.ts`（新增路由）

## 新增功能：用户中心任务详情查看
- 新增 `GET /account/api/tasks/:id` 后端接口，返回任务完整信息（含 input、output、error、metadata 等全部字段）
- 用户中心仪表盘"最近任务"表格和"任务管理"页面均新增"查看详情"按钮
- 点击后在任务行下方内联展开详情面板，展示：任务ID、类型、模型、状态、上游、Provider、输入参数、输出结果、错误信息、元数据等
- 支持再次点击收起，也可在详情面板内点击"收起"关闭
- 修改文件：`src/routes/account.ts`（新增 `handleGetAccountTask`、`publicTaskFull`，修改 `renderTasks`、`renderTaskDetail`、添加 `toggleTaskDetail` 及事件处理）
